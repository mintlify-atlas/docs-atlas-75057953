---
title: Axios class
description: Core class for creating Axios instances with customizable defaults and interceptors.
---

The `Axios` class is the core class that powers all Axios instances. It provides methods for making HTTP requests and managing request/response interceptors.

## Constructor

<ParamField path="instanceConfig" type="AxiosRequestConfig" optional>
  The default configuration for the instance. All requests made with this instance will merge their config with these defaults.
</ParamField>

```typescript
import { Axios } from 'axios';

const instance = new Axios({
  baseURL: 'https://api.example.com',
  timeout: 5000,
  headers: { 'X-Custom-Header': 'value' }
});
```

## Properties

### defaults

<ParamField path="defaults" type="AxiosDefaults">
  The default configuration object for this instance. You can modify defaults after instance creation.
</ParamField>

```javascript
const instance = new Axios();
instance.defaults.baseURL = 'https://api.example.com';
instance.defaults.headers.common['Authorization'] = 'Bearer token';
```

### interceptors

<ParamField path="interceptors" type="object">
  Interceptor managers for request and response interception.
</ParamField>

<ParamField path="interceptors.request" type="AxiosInterceptorManager<InternalAxiosRequestConfig>">
  Manager for request interceptors.
</ParamField>

<ParamField path="interceptors.response" type="AxiosInterceptorManager<AxiosResponse>">
  Manager for response interceptors.
</ParamField>

```javascript
// Add a request interceptor
instance.interceptors.request.use(
  (config) => {
    config.headers.Authorization = 'Bearer ' + getToken();
    return config;
  },
  (error) => Promise.reject(error)
);

// Add a response interceptor
instance.interceptors.response.use(
  (response) => response.data,
  (error) => {
    if (error.response?.status === 401) {
      // Handle unauthorized
    }
    return Promise.reject(error);
  }
);
```

## Methods

### request()

Dispatch a request with the given configuration.

<ParamField path="config" type="AxiosRequestConfig<D>" required>
  The configuration object for this request. Will be merged with instance defaults.
</ParamField>

<ParamField path="returns" type="Promise<R>">
  A promise that resolves to the response object.
</ParamField>

```typescript
instance.request<ResponseType>({
  method: 'post',
  url: '/user',
  data: { name: 'John' }
}).then(response => {
  console.log(response.data);
});
```

### getUri()

Generate a URL with parameters serialized according to the configuration.

<ParamField path="config" type="AxiosRequestConfig" optional>
  Configuration object with url, baseURL, and params.
</ParamField>

<ParamField path="returns" type="string">
  The computed URI with serialized parameters.
</ParamField>

```javascript
const uri = instance.getUri({
  url: '/users',
  params: { page: 1, limit: 10 }
});
console.log(uri); // '/users?page=1&limit=10'
```

## Usage example

```javascript
import { Axios } from 'axios';

// Create a custom instance
const api = new Axios({
  baseURL: 'https://api.example.com',
  timeout: 10000,
  headers: {
    'Content-Type': 'application/json'
  }
});

// Add request interceptor
api.interceptors.request.use((config) => {
  const token = localStorage.getItem('token');
  if (token) {
    config.headers.Authorization = `Bearer ${token}`;
  }
  return config;
});

// Add response interceptor
api.interceptors.response.use(
  (response) => response.data,
  (error) => {
    if (error.response?.status === 401) {
      // Redirect to login
      window.location.href = '/login';
    }
    return Promise.reject(error);
  }
);

// Make requests
api.get('/users').then(users => {
  console.log(users);
});

api.post('/users', { name: 'New User' }).then(user => {
  console.log('Created:', user);
});
```
