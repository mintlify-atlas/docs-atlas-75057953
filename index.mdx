# Promise-based HTTP client for the browser and Node.js

Axios is a powerful, promise-based HTTP client that works seamlessly in both browser and Node.js environments. With over 100 million weekly downloads, it's the most popular HTTP client for JavaScript applications.

## Why Axios?

Axios provides a simple, elegant API for making HTTP requests while handling complex scenarios like interceptors, automatic transforms, and request cancellation. It's the perfect tool for consuming REST APIs, uploading files, and managing HTTP communications in your JavaScript applications.

## Browser support

Axios supports all modern browsers including Chrome, Firefox, Safari, Opera, and Edge. The library uses XMLHttpRequest in browsers and the native http module in Node.js.

<Note>
  Axios requires a Promise polyfill for Internet Explorer and older browsers that don't support ES6 promises natively.
</Note>
